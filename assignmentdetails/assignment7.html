Assignment 7

WebGL

In this assignment, you will create a dual-layer drawing consisting of a two-dimensional background canvas and a three-dimensional foreground canvas.

Background Layer

The background layer of your drawing should consist of an HTML canvas with a 2D rendering context. The canvas should occupy the full width and height of the browser window, updated accordingly if the browser is resized. Continue to explore drawing techniques from recent weeks in your 2D canvas script, being sensitive to how processor-intensive it is; your background layer can be animated but this is not required. Because this is the background of your drawing, it should set the context for whatever you do in the foreground layer.

Foreground Layer

Using three.js, create a WebGL scene that animates 3D geometry in the foreground layer of your drawing. The object you create should be more complex than one of the three.js shape primitives. As such, you may choose to create your mesh with 3D graphics software, such as Blender or the three.js editor. It’s also possible to create more complex forms by combining shape primitives in three.js. A third option is to adapt from an existing 2D image or 3D model. The foreground layer should include at least one 3D model, material, lighting, and animation.

Submitting Your Assignment

Update the project link from your portfolio page and submit the following via NYU Classes.

  • The URL to your assignment in the general form of: i6.cims.nyu.edu/~netid/380/
  • A compressed archive of your project files (including the HTML, CSS, JavaScript, and any media resources)

Please note that you are required to submit files along with URLs in order to receive credit for your work.

Grading

This assignment is worth 10 points.

  • A new 2D canvas drawing layered behind the foreground (3 points)
  • The background layer should occupy the full browser window (1 point)
  • A custom 3D geometric object layered in front of the background (3 points)
  • Object material (1 point)
  • Lighting (1 point)
  • Basic animation (1 point)
