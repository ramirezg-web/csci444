Web API Assignment 5

The Blockchain API is the most trusted source for data on the Bitcoin Blockchain. It details Currency Statistics, Block Details, Mining Information, Network Activity, Wallet Activity, and Market Signals. Users can explore the total number of mined bitcoin that are currently circulating on the network, t he total USD value of bitcoin in circulation, the total number of transactions on the blockchain, and more. I think this is a unique way to conceptualize how fast and how popular web3/blockchain technology is, and how fast it is growing.

In this project, I leverage the change percent of BTC data to change how the page is experienced. The change percent variable changes the colors of items on the webpage, displays and hides media, and updates styles.
